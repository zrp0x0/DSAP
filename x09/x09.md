# 0x09 - BFS

## 알고리즘 설명

### Flood Fill
- 색 채우기

### BFS (Breadth First Search)
- 다차원 배열에서 각 칸을 방문할 때 **너비를 우선**으로 방문하는 알고리즘

---

## 예시

### 예제
- 목표: (0, 0)과 연결된 모든 파란색 칸 탐색
- 도구: Queue
- 1. 해당 지점을 방문했다는 표시를 남긴다
- 2. 큐에 해당 지점을 넣음
- 3. 큐의 front를 빼고 상하좌우를 확인
- 4. 파란색 칸이면서 방문하지 않은 경우 1번의 과정을 수행
- 5. 큐가 빌 때까지 탐색을 진행

### 동작 방식
- 1. 시작하는 칸을 큐에 넣고 방문했다는 표시를 남김
- 2. 큐에서 원소를 꺼내어 그 칸에 상하좌우로 인접한 칸에 대해 3번을 진행
- 3. 해당 칸을 이전에 방문했다면 아무것도 하지 않고, 처음으로 방문했다면 방문했다는 표시를 남기고 해당 칸을 큐에 삽입
- 4. 큐가 빌 때까지 2번을 반복
- **모든 칸이 큐에 1번씩 들어가므로 시간복잡도는 칸이 N개일 때 O(N)**

### 자주 사용하는 STL
- utility 헤더에 있는 pair
```cpp
pair<int, int> p1 = make_pait(10, 1);
pair<int, int> p2 = {10, 1}; // C++ 11

cout << p1.first << ' ' << p1.second;
```
    - 크기 비교도 내부적으로 구현되어있어, first를 먼저 비교하고, second를 비교함

- BFS 정석적인 구현
```cpp
#include <bits/stdc++.h>
using namespace std;

#define X first
#define Y second

int board[502][502]; // 입력받는 맵
bool vis[502][502]; // 방문 여부 확인

int dx[] = {1, 0, -1, 0}; // 상하좌우
int dy[] = {0, 1, 0, -1}; // 상하좌우

int N = 7, M = 10; // 행, 열

int main(void) {
    ios_base::sync_with_stdio(false);
    cin.tie(nullptr);

    queue<pair<int, int>> Q;

    vis[0][0] = 1; // 시작 지점을 방문 표시를 남김
    Q.push({0, 0}); // 시작 지점을 큐에 넣음

    while (!Q.empty()) {
        pair<int, int> cur = Q.front();
        Q.pop();

        for (int i = 0; i < 4; i++) {
            // 상하좌우 확인
            int nx = cur.X + dx[i]; 
            int ny = cur.Y + dy[i];

            // out of bound
            if (nx < 0 || nx >= N || ny < 0 || ny >= M) continue;

            // 이미 방문했거나 갈 수 없는 곳
            if (vis[nx][ny] || board[nx][ny] != 1) continue;

            // 방문하지않고 갈 수 있는 곳 처리
            vis[nx][ny] = 1;
            Q.push({nx, ny});
        }
    }

    return 0;
}
```

### 자주 실수하는 경우
- 시작점에 방문했다는 표시를 남기지 않음
- 큐에 넣을 때 방문했다는 표시를 하는 대신 큐에 빼낼 때 방문했다는 표시를 남김
    - 인접해있는 경우, 같은 칸을 여러 번 넣는 상황이 발생할 수 있어 시간초과가 날 수 있음
- 이웃한 원소가 범위를 벗어났는지에 대한 체크를 잘못함

## BFS 기본 예제

### Intuition
- 상하좌우로 연결된 그림의 크기를 알아내기
- 도화지에 있는 모든 그림 찾아내기

### Approach
- 새로운 시작점을 발견하면 그림의 개수를 1증가
- 새로운 시작점에 해당하는 BFS를 하면서 Q.push()를 하는 횟수로 그림의 넓이를 구함
- 구한 그림의 넓이를 ans랑 비교 후 크면 ans를 갱신

### Complexity
- Time complexity: O(N * M)
    - while 문은 특정 조건일 때만 실행되고, 아무리 많아도 N*M번만 실행됨
    - O(N x M) + O(N x M) 이므로 O(N x M)
    - N, M의 조건이 500이하이므로 최대 약 O(N^3) 안에 문제를 풀어야함 -> 통과 가능

- Space complexity: O(N * M)
    - 128MB이므로 무난하게 통과 가능
    - board / vis => 250000 + 250000 = 500000개 < 33554432

### Code
```cpp []
// BOJ 1926번: 그림

#include <bits/stdc++.h>
using namespace std;

#define X first
#define Y second

int board[502][502];
int vis[502][502];

int dx[] = {1, 0, -1, 0};
int dy[] = {0, 1, 0, -1};

int cnt = 0;
int ans = 0;

int main(void) {
    ios_base::sync_with_stdio(false);
    cin.tie(nullptr);

    int N, M;
    cin >> N >> M;

    queue<pair<int, int>> Q;

    for (int i = 0; i < N; i++) {
        for (int j = 0; j < M; j++) {
            cin >> board[i][j];
        }
    }

    for (int x = 0; x < N; x++) {
        for (int y = 0; y < M; y++) {
            if (board[x][y] == 1 && vis[x][y] == 0) {
                cnt++;
                vis[x][y] = 1;
                Q.push({x, y});

                int size = 0;

                while (!Q.empty()) {
                    pair<int, int> cur = Q.front();
                    Q.pop();
                    size++;

                    for (int i = 0; i < 4; i++) {
                        int nx = cur.X + dx[i];
                        int ny = cur.Y + dy[i];

                        if (nx < 0 || nx >= N || ny < 0 || ny >= M) continue;
                        if (vis[nx][ny] > 0 || board[nx][ny] != 1) continue;

                        vis[nx][ny] = 1;
                        Q.push({nx, ny});
                    }
                }

                if (ans < size) {
                    ans = size;
                }
            }
        }
    }

    cout << cnt << '\n';
    cout << ans << '\n';

    return 0;
}
```

## 응용 1 - 거리 측정

### Intuition
- 다차원 배열에서의 최단 거리를 구하는 문제
- 시작점에서 끝점으로 가는 문제
- 즉, 시작점에서 끝점까지의 최단 거리를 구하는 문제

### Approach
- 다차원 배열을 보자마자 BFS를 사용해야겠다고 생각
- BFS는 각 지점으로부터 최단 거리를 구하는데 사용할 수 있음
- BFS의 가장 중요한 성질 => 한 정점을 방문하고 다음 정점으로 넘어갈 때의 거리가 매번 1차이
- 즉 방문했다말고, 시작지점으로부터 얼마나 떨어져있는지 구하는 dist 배열을 두고 출발한 정점의 값 + 1을 통해 거리를 갱신

### Complexity
- Time complexity: O(N x M)

- Space complexity: O(N x M)

### Code
```cpp []
// BOJ 2178번: 미로 탐색

#include <iostream>
#include <queue>
#include <utility>
using namespace std;

#define X first
#define Y second

int board[102][102];
int dist[102][102];

int dx[] = {1, 0, -1, 0};
int dy[] = {0, 1, 0, -1};

int main(void) {
    ios_base::sync_with_stdio(false);
    cin.tie(nullptr);

    int N, M;
    cin >> N >> M;
    for (int x = 0; x < N; x++) {
        string str;
        cin >> str;
        for (int y = 0; y < M; y++) {
            board[x][y] = str[y] - '0';
        }
    }

    queue<pair<int, int>> Q;
    Q.push({0, 0});
    dist[0][0] = 1;

    while (!Q.empty()) {
        pair<int, int> cur = Q.front();
        Q.pop();

        for (int k = 0; k < 4; k++) {
            int nx = cur.X + dx[k];
            int ny = cur.Y + dy[k];

            if (nx < 0 || nx >= N || ny < 0 || ny >= M) continue;
            if (dist[nx][ny] > 0 || board[nx][ny] == 0) continue;

            dist[nx][ny] = dist[cur.X][cur.Y] + 1;
            Q.push({nx, ny});
        }
    }

    cout << dist[N - 1][M - 1] << '\n';

    return 0;
}
```

## 응용 2 - 시작점이 여러 개일 때 (다중(동시) 시작점)

### Intuition
- 상하좌우의 토마토가 익는 시간은 익은 토마토가 있는 지점에서 +1을 하면 됨
- 그 이후로도 똑같이 반복
- BFS를 사용하여 거리를 계산 == 시간을 계산

### Approach
- 상자에서 -1로 표시된 곳은 갈 수 없음
- 또한 -1로 둘러싸여있는 곳은 평생 익지 않음
- BFS를 돌릴 때마다 방문한 경우 board에 방문했다는 표시를 남김
- 나중에 board를 순회하면서 0이 남아있다는 것은 익지 않은 토마토가 있다는 표시
- 방문을 했거나 벽이 있으면 BFS를 하지 않음

### Complexity
- Time complexity: O(N x M)

- Space complexity: O(N x M)

### Code
```cpp []
// BOJ 7576번: 토마토

#include <bits/stdc++.h>
using namespace std;

#define X first
#define Y second

int N, M;
int ans;

int board[1002][1002];
int dist[1002][1002];

int dx[] = {1, 0, -1, 0};
int dy[] = {0, 1, 0, -1};

int main(void) {
    ios_base::sync_with_stdio(false);
    cin.tie(nullptr);

    cin >> M >> N;

    queue<pair<int, int>> Q;

    for (int x = 0; x < N; x++) {
        for (int y = 0; y < M; y++) {
            cin >> board[x][y];
            dist[x][y] = -1;
            if (board[x][y] == 1) {
                Q.push({x, y});
                dist[x][y] = 0;
            }
        }
    }

    while (!Q.empty()) {
        pair<int, int> cur = Q.front();
        Q.pop();

        for (int k = 0; k < 4; k++) {
            int nx = cur.X + dx[k];
            int ny = cur.Y + dy[k];

            if (nx < 0 || nx >= N || ny < 0 || ny >= M) continue;
            if (dist[nx][ny] > -1 || board[nx][ny] == -1) continue;

            dist[nx][ny] = dist[cur.X][cur.Y] + 1;
            ans = dist[nx][ny];
            board[nx][ny] = 1;
            Q.push({nx, ny});
        }
    }

    for (int x = 0; x < N; x++) {
        for (int y = 0; y < M; y++) {
            if (board[x][y] == 0) {
                cout << -1 << '\n';
                return 0;
            }
        }
    }

    cout << ans << '\n';

    return 0;
}
```	

### BFS의 성질 (아직 완전히 이해하지 못함)
- 큐에 들어가는 순서는 거리가 1씩 차이가 나는 걸 먼저 다 넣음
- 즉, 큐에 쌓이는 순서는 거리 순!!

## 응용 3 - 시작점이 두 종류일 때

### Intuition
- 시간(거리)를 측정해야하고, 각 칸의 상하좌우로 움직일 수 있기 때문에 일단 BFS를 떠올림

### Approach
- 정말 핵심
    - 불이 갇혀있는 경우가 있을 수도 있음
    - **불은 지훈이에게 영향을 주지만 지훈이는 불에게 영향을 주지 않음**
        - 둘이 서로 영향을 주면 하나를 먼저하면 안됨 -> 나중에 나온다고 하네요
        - 즉, 불을 먼저 BFS를 
        
- 두번째로 out of bound가 탈출 조건임
- 또한 불이 완전히 이동하지 못하는 경우가 있음 <= 이게 생각하기 어려움
    - 무슨 말이냐면 단순히 지훈 >= 불이면 continue를 하면 안됨
        - 불이 벽에 둘러쌓여있어서 지훈이가 가는 깅에 안 번지면 -1 상태 지훈이 1에 가도 1 >= -1이므로 continue가 실행됨

### Complexity
- Time complexity: O(R x C)
 
- Space complexity: O(R x C)

### Code
```cpp []
// BOJ 4179번: 불!
#include <bits/stdc++.h>
using namespace std;

#define X first
#define Y second

int R, C;

char board[1002][1002];
int distJ[1002][1002];
int distF[1002][1002];

int dx[] = {1, 0, -1, 0};
int dy[] = {0, 1, 0, -1};

int ans;

int main(void) {
    ios_base::sync_with_stdio(false);
    cin.tie(nullptr);

    queue<pair<int, int>> JQ;
    queue<pair<int, int>> FQ;

    cin >> R >> C;
    for (int x = 0; x < R; x++) {
        for (int y = 0; y < C; y++) {
            cin >> board[x][y];
            distJ[x][y] = -1;
            distF[x][y] = -1;
            if (board[x][y] == 'J') {
                JQ.push({x, y});
                distJ[x][y] = 0;
            } 
            if (board[x][y] == 'F') {
                FQ.push({x, y});
                distF[x][y] = 0;
            }
        }
    }

    while (!FQ.empty()) {
        pair<int, int> cur = FQ.front();
        FQ.pop();

        for (int k = 0; k < 4; k++) {
            int nx = cur.X + dx[k];
            int ny = cur.Y + dy[k];

            if (nx < 0 || nx >= R || ny < 0 || ny >= C) continue;
            if (distF[nx][ny] > -1 || board[nx][ny] == '#') continue;

            distF[nx][ny] = distF[cur.X][cur.Y] + 1;
            FQ.push({nx, ny});
        }
    }

    while (!JQ.empty()) {
        pair<int, int> cur = JQ.front();
        JQ.pop();

        for (int k = 0; k < 4; k++) {
            int nx = cur.X + dx[k];
            int ny = cur.Y + dy[k];

            if (nx < 0 || nx >= R || ny < 0 || ny >= C) {
                cout << distJ[cur.X][cur.Y] + 1 << '\n';
                return 0;
            }
            if (distJ[nx][ny] > -1 || board[nx][ny] == '#') continue;
            if (distJ[cur.X][cur.Y] + 1 >= distF[nx][ny] && distF[nx][ny] != -1) continue;
            
            distJ[nx][ny] = distJ[cur.X][cur.Y] + 1;
            JQ.push({nx, ny});
        }
    }

    cout << "IMPOSSIBLE" << '\n';

    return 0;
}
```

## 응용 4 - 1차원에서의 BFS

### Intuition
- 상하좌우에 너무 국한되지 않고, 특정칸을 동시에 나아가면서 최단 거리를 구하고 싶을 때 
- BFS를 사용하면 됨

### Approach
- 최단 거리를 구하는 문제이기 때문에 결국 일단 BFS를 고려
- 양옆 점프를 1 -1 *2씩 해가면서 각 거리를 빠르게 전체적으로 탐색함
- 주의해야할 점
    - 있는 구간이 0 <= <= 100000이라고 했지 넘아가면 안되다는 것은 없음
    - 다행히도 이 문제에서 음수로 가는건 정말 비효율적(+1만 해야함)이고, 10만을 넘어가는 행위도 비효율적임 (-1만 해야함)
    - 조금 신경쓴 범위는 0 ~ 10만이기에 범위를 100001 + 1로 배열을 생성
    - 넘어가는 범위는 100000보다 큰 부분을 넘어가면 out of bound
    - 또한 맵은 딱히 필요없이 -1로 한 배열을 초기화하고 각 BFS 거리를 담아두고 나중에 최종적으로 K의 위치에 있는 값을 뽑아옴

### Complexity
- Time complexity: O(M) | 0 <= M <= 100000

- Space complexity: O(M)

### Code
```cpp []
// BOJ 1697번: 숨박꼭질
#include <bits/stdc++.h>
using namespace std;

int N, K;

int vis[100002];

int dx[] = {1, -1, 2};

int main(void) {
    ios_base::sync_with_stdio(false);
    cin.tie(nullptr);

    for (int i = 0; i < 100002; i++) {
        vis[i] = -1;
    }

    queue<int> Q;

    cin >> N >> K;

    Q.push(N);
    vis[N] = 0;

    while (!Q.empty()) {
        int cur = Q.front();
        Q.pop();

        for (int i = 0; i < 3; i++) {

            int nx = 0;
            if (i == 2) {
                nx = cur * dx[i];
            } else {
                nx = cur + dx[i];
            }

            if (nx < 0 || nx > 100000) continue;
            if (vis[nx] > -1) continue;

            Q.push(nx);
            vis[nx] = vis[cur] + 1;
        }
    }

    cout << vis[K] << '\n';

    return 0;
}
```
